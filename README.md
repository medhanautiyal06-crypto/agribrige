# AgriBridge

Bridging Farmers with Smart Agriculture 🌱🚜
Empowering farmers with AI-powered disease detection, smart crop recommendations, weather insights, and multilingual assistance.
AI-powered agriculture platform for farmers.

## Features

- Authentication
- Farmer Dashboard
- Disease Detection UI
- AI Chatbot UI
- Crop Recommendation System

Built using React + Vite + Tailwind CSS.
